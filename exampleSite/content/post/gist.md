As of Hugo v0.15, Github Gists can be embedded in a page with a builtin
shortcode. The shortcode 
```
  {{</* gist shawnohare bc528b61820d6b7136c2 */>}}
```
produces:
{{< gist shawnohare bc528b61820d6b7136c2 >}}
